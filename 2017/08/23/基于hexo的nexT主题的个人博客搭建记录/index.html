前言

搭建过程中睬坑记录

安装hexo的时候报错

问题描述：
使用命令安装hexo（sudo npm install -g hexo）的时候报错了：

解决办法：
npm config set unsafe-perm=true
貌似是因为权限问题，使用以上命令后可以安装成功了。
参考：(https://github.com/Microsoft/BashOnWindows/issues/14)

you can fix that error by allowing unsafe perms

集成搜索功能的时候报错

问题描述：
执行hexo algolia时报Error AlgoliaSearchNodeJS.AlgoliaSearchCore
报错信息:

解决办法：
安装hexo-algolia@0.2.0，再执行hexo algolia就可以了
（注意站点目录下的package.json中hexo-algolia版本也要改成^0.2.0）参考：(http://leunggeorge.github.io/2017/07/09/%E6%89%A7%E8%A1%8Chexo%E6%97%B6%E7%9A%84%E4%B8%80%E4%BA%9B%E6%8A%A5%E9%94%99%E5%A4%84%E7%90%86/)

待续。
